# TETRAD ‚ö°

<h2>Introduction:</h2>
<em>App Link:  https://tetrad-3114.herokuapp.com/ </em>
<p>In this fully responsive web-app the user has to first create a account or he can sign up
using Google or Facebook too . After creating an account the user can sign in using same email again.Once account setup is done ,User can access the app 
features which are enlisted below:<ul>
  <li><h3>KEEPER APP üìù</h3><p>This <strong>MERN</strong> app first asks for user's unique id/name and once logged in the user is able to keep his notes
      in a private space which only the user with specific unique name/id can access.The data is backed up on Atlas Cluster.The user can add as well
    as delete notes in the Keeper App.</p></li>
  <li><h3>SIMON GAME üéÆ</h3><p>This is a implementation of the Simon Game,All the processing and the backend stuff is handled using Javascript,jQuery.
    If you want to know more about Simon Game, refer this video https://www.youtube.com/watch?v=1Yqj76Q4jJ4 </p></li>
  <li><h3>WEATHER FORECAST ‚õàÔ∏è</h3><p>This app lets the user know the weather for four consecutive days, 
    The user just has to enter the city name and the weather will be displayed.</p></li>
  <li><h3>BAND ü•Å</h3><p>If you are a rock band enthusiast, well then we got you covered ! This app lets the user play band using keyboard.What for 
    phone users ? There's room for everyone they can play instruments by simply tapping on them . Head over to Tetrad to feel the Music.</p></li>
  <li><h3>CONTRIBUTEüí™ </h3><p>Want to be a part of the community ? Head over to Contribute and sign up with your email and we will 
    reach out to you !</p></li>
  <li><h3>LOG OUT üëã</h3><p>As the name suggests you will be logged out the <em>TETRAD APP</em>.
</ul>
